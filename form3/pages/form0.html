O atributo type = "text" especifica o valor que o input reveb

Type - "button" cria um botão para um evento com JS, por exemplo.

O type = "submit" é utilizado para envio de dados, sem necessidade de JS. Envia pelo protocolo 'http'.

O checkbox é usado para dar mais de uma opção para ousuário.

A type "date" gera um calendário que facilita a inserção de data.



O type = "email" identifica se o campo foi preenchido com "@" e ".". Não faz validação.

O type = "type" é usado para envio de arquivo,

O type = "number" é usado para determinar o valor recebido no input como número.

O type = "password" protege os dados de digitados.

O type = "radio" é usado para dar apenas uma opção para o usuário.

O type = "range" gera uma opção dinânica partindo "A - B ou 0 - 10".

O type = "search" gera um campo de busca dentro do teu site.

O type = "tel" usado para entrada de valor numérico.
